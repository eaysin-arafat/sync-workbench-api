Get Started

    API Endpoint

        https://api.sync-workbench.com/

The Sync-Workbench API provides programmatic access to read and manage employee data. You can retrieve employee details, manage roles, track performance, and more.

To use this API, you need an API key. Please contact us at support@sync-workbench.com to get your own API key.

Get Employees


# Example to get employees:
curl \
-X POST https://api.sync-workbench.com/employee/get \
-F 'api_key=your_api_key' \
-F 'status=active' \
-F 'offset=0' \
-F 'limit=50'

To get employee details, you need to make a POST call to the following URL: https://api.sync-workbench.com/employee/get 



Result example:

{

Query Parameters

Field Type Description
api_key String Your API key.
status String (optional) A status to filter employees by. E.g., active, inactive.
page Integer (optional - default: 1) The page number for pagination.
limit Integer (optional - default: 10) The number of records to return, between 1 and 100.
sort_type String (optional) The type of sorting. Values: 'asc' or 'desc'.
sort_by String (optional) The field to sort by (e.g., 'updatedAt', 'firstName').
populate String (optional) The related property to populate, e.g., 'location'.
role String (optional) The role to filter employees by, e.g., 'Developer'.

Errors

The Sync-Workbench API uses the following error codes:

Error Code Meaning
MISSING_REQUIRED_PARAMETERS The request is missing required parameters.
INVALID_API_KEY The API key provided is invalid.
EXPIRED_API_KEY The API key provided has expired.
INVALID_QUERY_PARAMETERS The query parameters provided are invalid.
SYNTAX_ERROR Check for syntax errors like missing quotes or extra commas.
BAD_REQUEST Validation failed for the input data.
INTERNAL_SERVER_ERROR An unexpected error occurred on the server.
NOT_FOUND The requested resource was not found on this server.
AUTHENTICATION_FAILED Authentication failed due to invalid credentials.
PERMISSION_DENIED You do not have permission to access this resource.
RESOURCE_CONFLICT A resource with the specified value already exists.
UNPROCESSABLE_ENTITY The request could not be processed due to semantic errors.
METHOD_NOT_ALLOWED The HTTP method used is not allowed for this endpoint.
TOO_MANY_REQUESTS Too many requests were sent in a short period.
RESOURCE_NOT_FOUND The requested resource with a specific ID was not found.
DUPLICATE_KEY_ERROR A resource with the same key already exists.
